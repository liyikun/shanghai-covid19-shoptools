# 团购助手

辅助微信群接龙生成报价单

    本人不是 python 程序员，现学现用，写得烂别喷我。。。

## 群话术模版

```
商品价格
[楼号][房间][昵称](可乐:70)(雪碧:80)(牛肉:100)
接龙格式是
[楼号][房间][昵称](套餐1*份数)(套餐2*份数)
例如
[2][1501][大喵](3*1)(2*2)
```

## 如何使用

在根目录下写入 require.txt
```
[楼号][房间][昵称](可乐:70)(雪碧:80)(牛肉:100)
[2][1501][大喵](3*1)(2*2)
[2][1303][Lexie](2*1)
[2][1402][Jessica](1*1)
```

然后运行 (需要 python3)

```
python main.py
```

根目录会出来一个 result.xlsx，然后就可以导入什么云文档分享了。
